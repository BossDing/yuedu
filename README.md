#悦读
项目介绍：项目为个人练习开发，是一个用户登录后可以发布、阅读的平台。

#技术栈：
前端主要技术使用了uni-app，前端实现了微信登录，下拉加载，上拉刷新等功能

后端采用了springboot进行开发，为前端提供所有接口，数据库使用了mysql。
